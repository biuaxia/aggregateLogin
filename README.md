# SpringBoot 集成彩虹聚合登录示例代码

## 硬编码说明

### app.config

请在 `src/main/resources/app.config` 路径下新建此文件，内容为：

```properties
# 应用详情-APPID
appID=换成你自己的
# 应用详情-APPKEY
appKey=换成你自己的
# 跳转回本系统的地址
redirectUrl=http://biuaxia.top/api/login/callback
```
